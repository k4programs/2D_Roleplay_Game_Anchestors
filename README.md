# Projekt Sonnenwende-Gefährten - Beta-Test

Vielen Dank für deine Teilnahme am Beta-Test von "Projekt Sonnenwende-Gefährten"!

## Installation

1.  Lade die neueste Beta-Version von der [Releases-Seite](https://github.com/k4programs/2D_Roleplay_Game_Anchestors/releases) herunter.
2.  Entpacke die ZIP-Datei in einen Ordner deiner Wahl.
3.  Starte das Spiel, indem du die `ProjektSonnenwendeGefaehrten.exe`-Datei ausführst.

## Was zu testen ist

Bitte teste alle Aspekte des Spiels, insbesondere:

*   **Kampfsystem:** Ist der Kampf ausbalanciert? Fühlt er sich fair an?
*   **Crafting-System:** Sind die Rezepte sinnvoll? Ist das Crafting-System einfach zu bedienen?
*   **Dialoge und Quests:** Sind die Dialoge ansprechend? Funktionieren die Quests wie erwartet?
*   **Allgemeines Gameplay:** Gibt es irgendwelche Bugs oder Probleme, die das Spielerlebnis beeinträchtigen?
